# Akinator
## Projeto de Estrutura de Dados Avançado

Um protótipo simples de um jogo estilo akinator usando estruturas de dados avançadas como DFS e BFS.

### Tecnologias Usadas
Python 3.10+

### Estrutura do Projeto
* main.py:
    Ponto de entrada do programa. Responsável por iniciar o jogo, gerenciar o fluxo principal e integrar os módulos.

* bfs.py:
    Implementação da busca em largura (Breadth-First Search). Utilizada para percorrer a árvore de decisões de forma nível a nível.

* dfs.py:
    Implementação da busca em profundidade (Depth-First Search). Utilizada para explorar a árvore de decisões até seus ramos mais profundos.

* node.py:
    Define a estrutura básica de um nó da árvore. Contém os dados (pergunta ou resposta) e referências para os filhos.

* tree.py:
    Implementa a estrutura da árvore de decisões. Responsável por inserir, organizar e manipular os nós.

* game.py:
    Contém a lógica do jogo estilo Akinator. Gerencia perguntas ao usuário, decisões baseadas nas respostas e aprendizado simples


### Execução do Projeto

Baixar os arquivos e executar o main.py, nenhuma importação externa requerida.
